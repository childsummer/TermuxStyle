# README

Make your Termux colorful!!

## Use：
```shell
sh -c "$(curl -fsSL https://github.com/Cabbagec/termux-ohmyzsh/raw/master/install.sh)"
```

## Set color：
Run`chcolor`change color, or：
```shell
~/.termux/colors.sh
```

## Set font:
Run`chfont`change fonts, or：
```shell
~/.termux/fonts.sh
```

## Need package：
 - curl
