# 讀我

讓你的Termux更多采多姿！
介面已經繁體中文化，對臺灣人更方便！
原本的作者是GitHub上的oh-my-termux，
自訂的功能是GitHub上的fsteam-cyber-indonesia加的，
然後是由我（yuhuan-afk）中文化的～

## 用法：
```shell
cd
git clone https://github.com/yuhuan-afk/TermuxStyle
cd TermuxStyle
bash install.sh
```

## 設定配色：
使用`chcolor`命令更改配色，或者：
```shell
~/.termux/colors.sh
```
也行

## 設定字體:
使用`chfont`命令更改字體，或者：
```shell
~/.termux/fonts.sh
```
也行
